Condition Monitoring of Induction Motor Using Vibration Signals and Machine Learning Classification

Location: Technische Universität Dresden, Dresden, Germany

Duration: Summer 2023

Role: Mechatronics Engineering Researcher

Project Description: Led the development and execution of research protocols aimed at enhancing the reliability and performance of electric motors using advanced analytical and machine learning techniques.

  • Designed experimental setups using SolidWorks to optimize weight distribution and ensure experimental accuracy.
  • Collaborated with cross-functional teams to ensure comprehensive data collection and robust analysis.
  • Developed software tools in Node.js and Node-Red for data processing and user interface improvements, enhancing operational efficiency.
  • Created a pioneering machine learning framework in Python to detect early signs of electric motor failure from vibration data.
  • Contributed to building a predictive model that classifies motor failures, significantly improving reliability and performance metrics.
  • Presented findings at the NCUR conference in Los Angeles on "Condition Monitoring of Induction Motor Using Vibration Signals and Machine Learning Classification".

Drone Monitoring and Control (Swarm)

Location: Cooper Union, New York, NY

Duration: Fall 2021 - Spring 2022

Role: Student Researcher

Project Description: Implemented a drone swarm control system using ROS, Linux, and Python, enabling synchronized operation of multiple drones for collaborative tasks.

  • Utilized Vicon cameras and open-source drone technology to monitor and analyze the motion of the drone swarm.
  • Developed algorithms for autonomous decision-making and obstacle avoidance.
  • Performed maintenance and updates on drone hardware and software.
