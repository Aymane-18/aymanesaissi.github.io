Autonomous Mobile Robot Research Project

Location: Cooper Union, New York, NY

Duration: Fall 2024

Project Overview

This project involves the design and development of an autonomous mobile robot intended for competitive robotics applications. The robot is equipped with sensors and actuators to navigate a dynamic environment, avoid obstacles, and complete specific tasks such as object detection and ball launching. The goal is to optimize the robot's performance in mobility, accuracy, and reliability under various operational conditions.

Skills Used

  • CAD Design with SolidWorks and Autodesk
  • Arduino programming for robotics control
  • Sensor integration (Ultrasonic, IR sensors)
  • 3D printing for custom components
  • Motor control with L298N motor driver
  • System integration for efficient communication between sensors and actuators

System Design and Media

Project Description

The autonomous robot is designed with a clear acrylic frame and is powered by 12V brushed DC motors. The robot features an array of ultrasonic and infrared sensors for obstacle detection and environmental awareness. The central task of the robot is to launch ping-pong balls using a spring plunger mechanism, activated by a solenoid. A custom 3D-printed barrel is used to launch the balls with precision. The robot is controlled by an Arduino UNO, which processes sensor data and sends commands to the motors and solenoid for action.

Key Contributions

  • Designed the robot frame using SolidWorks and fabricated components using laser cutting and 3D printing techniques.
  • Developed the drive system utilizing custom motor mounts and high-traction wheels for stable movement.
  • Integrated a sensor array, consisting of ultrasonic and IR sensors, to enable obstacle avoidance and target detection.
  • Programmed the Arduino to handle sensor input, control the drive motors, and trigger the ping-pong ball launcher.
  • Created a sensor mount system to reduce external clutter and ensure reliable sensor readings.
  • Tested the robot's functionality in various scenarios to ensure robustness and reliability in the field.

Open Channel Flow Research Project

Location: Cooper Union, New York, NY

Duration: Fall 2023 - Summer 2024

Role: Mechanical Engineering Researcher

Project Overview

Conducted research on fluid dynamics, focusing on both liquid and gas flows within open channel systems, with applications to aerospace engineering, particularly rocket propulsion. This involved advanced computational modeling, theoretical analysis, and real-world experimental validations.

Skills Used

  • CAD Design with SolidWorks and Autodesk
  • ANSYS for fluid dynamics simulations
  • Python for data visualization and analysis
  • Experimental design and high-speed imaging analysis
  • Shock wave theory and Mach number analysis
  • Textbook editing for aerospace and fluid dynamics

ANSYS Simulations and Media

Project Description

In this project, we developed specialized nozzles for open channel flow systems and analyzed how different geometries affect fluid behavior and shock wave propagation. Using both experimental and computational tools, we aimed to optimize nozzle design for aerospace applications, particularly rocket propulsion systems.

Key Contributions

  • Designed various nozzle geometries in SolidWorks to study fluid behavior, shock wave formation, and flow efficiency.
  • Applied ANSYS simulations to visualize and analyze the interaction of liquid and gas flows with the nozzle designs.
  • Analyzed flow patterns and turbulence using high-speed cameras and Python-based data visualization techniques.
  • Validated theoretical models through experimentation, capturing real-time shock wave propagation using pressure sensors and high-speed imaging.
  • Edited textbooks to integrate research findings into broader aerospace and fluid dynamics applications.

Tracking Strain in Stress Concentration Using Digital Image Correlation

Location: Cooper Union, New York, NY

Duration: Spring 2024

Role: Mechanical Engineering Researcher

Project Overview

This project focuses on experimental stress analysis using Digital Image Correlation (DIC) to validate theoretical stress concentration data. It aims to track strain distribution around stress concentration points, such as holes in engineering components, using high-resolution imaging and DIC software.

Skills Used

  • SolidWorks for CAD and FEM analysis
  • MATLAB for data processing and contour plotting
  • Digital Image Correlation (DIC) techniques
  • Instron testing for tensile loading
  • Experimental setup and data collection using VIC-3D software

Project Description

In this project, the team used DIC technology to analyze the strain distribution around stress concentration areas in aluminum samples. The samples were tested in a tensile machine while being observed with a high-resolution stereo camera. The strain data was compared with theoretical and simulation results to identify deviations and optimize experimental procedures.

Media and Results

Key Contributions

  • Conducted experimental tests using aluminum 6061 samples and applied speckle patterns for DIC.
  • Used Instron tensile machine for sample loading while recording strain data through VIC-3D software.
  • Processed strain data in MATLAB to create detailed contour plots of stress concentration.
  • Compared experimental results with theoretical models and FEM simulations to validate findings.
